## Welcome to Pagsubay Research Frontend Repository
This repository houses the frontend composition of the Epicenter Estimation Project created by the Pagsubay Research Team

## File structure:
-Create main folder in your device(pagsubay-frontend)
<br>
-Create host file for every ticket(e.g. pagsubay-#26-repoSetup)

## Cloning process:
-> Copy repo link -> VC Code -> Open Ticket host file -> Terminal -> run command "git clone (repo link)" -> cd pagsubay-frontend -> npm i

## Branch naming:
Feat: features that you've added to the porject
Chore: Tasks that you've done for the project 
Fix: bug fixes or for QA tickets
Refactor: Code improvements and feature fixing

e.g. 
feat/1--repoSetup
chore/1--changedTheBackgroundColor
fix/1--fixedImageAlignment
refactor/1--improvedLandingPageResponsiveness

## Commit messages:
feat: commit message
fix: commit message
refactor: commit message
chore: commit message

## Commits Guidelines:
Commit often but meaningful
Small, logical commits to make it easier to review, understand, and roll back changes if needed.
Always start the commit message with a verb:
## feat: Add/Change/Made/Remove something
